Leaflet OS Map Demo

This is a non full screen demo in order to minimise number of tiles downloaded from map providers. The version in the Github src folder will run full screen in a browser, but you will have to obtain your own API keys.



Check out the following functionality:

  • Seamless zoom - OS data 1:25,000 at high zoom, OS 1:50,000 at mid zoom, MapBox data at low zoom.
  • Autolocate button will zero in on your current location, and will track your movement if using a mobile phone
  • Configurable Place lists. You can add your own json places file to the places folder. Displays locations as cluster of markers.
  • Click map to show current location as lon, lat and a OS Grid Ref. These can be copied to the clipboard.
  • Search button.
    • If a places list has been selected, searches locations in that places list.
    • If no places list has been selected, does one of three searches depending on search string:
        [lat],[lon] e.g. 55.466,-3.515 does latlon search.
        [OS GridRef] e.g.NY442917 does gridref search.
        Otherwise uses the Ordnance Survey place names gazeteer.
  • Measurment button. Allows lines an paths to be measured on the map.
